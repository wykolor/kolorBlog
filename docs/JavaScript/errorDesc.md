# 错误信息描述

| Error.name     | 错误信息描述             |
| -------------- | ------------------------ |
| EvalError      | eval()的使用和定义不一致 |
| RangeError     | 数值越界                 |
| ReferenceError | 非法或不能识别的引用数值 |
| SyntaxError    | 发生语法 解析错误        |
| TypeError      | 操作数类型错误           |
| URIError       | URI 处理函数使用不当     |
