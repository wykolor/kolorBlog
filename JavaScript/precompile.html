# 预编译

js 运行三部曲

    1. 语法分析：全篇扫描整个文档，分析语法；有错报错,默认添加分号，划分js语句这一步都开始执行，程序如果有语法错误，直接报错 不再执行下面的阶段。
    2. 预编译阶段：------>函数声明整体提升 变量 声明提升 发生在函数执行的前一刻。
    3. 解释执行：解释一句执行一句。
