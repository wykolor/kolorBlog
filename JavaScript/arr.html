    # 数组

    创建数组

    1. 字面量方式：var arr = [];
    2. 利用系统自带的构造函数 var arr = new Array();

    两种方式唯一的差别就是，当构造函数传参一位的时候，表示创建的是一个长度为 X 的空数组，而字面量则表示创建了一个长度为 1，数组第 0 位是 X 的数组

    数组的方法

    • 改变原数组：push() pop() shift() unshift() sort() revese() splice()
    • 不改变原数组：concat() join() slice() ----> split() toString()

    改变原数组

    # 1. sort() 给数组排序

     /**
 *   可以接收一个函数作为参数让开发者定义排序规则
 *  1、函数必须传入两个参数
 *  2、看函数返回值 1) 返回正数，后面的值放在前面
 *                  2) 返回负数， 前面的数放在前面，保持不动
 *                  3) 返回0，保持不动
 */
    var arr = [1,5,82.30,51,34,3,2];
    arr.sort(function(prop1,prop2){
        return prop1 - prop2;  // 升序
        return prop2 - prop1;  // 降序
    })

    push()练习题

    第 01 题. 手写 push 方法

       Array.prototype.push = function() {
       for (var i = 0; i < arguments.length; i++) {
           this[this.length] = arguments[i];
       }
       return this.length;
   };

    # 数组去重

     <!--数组去重-->
     Array.prototype.unique = function() {
        var obj = {},
            arr = [],
            len = this.length;
        for (var i = 0; i < len; i++) {
            if (!obj[this[i]]) {
                obj[this[i]] = "unique";
                arr.push(this[i]);
            }
        }
        return arr;
    };
